PPHlo reference
===============

PPHlo is short for (PPU High level ops), it's the assembly language of PPU.

PPHlo is build on `MLIR <https://mlir.llvm.org/>`_ infrastructure, the concrete ops definition could be found :doc:`here<../../ppu/dialect/pphlo_ops.td>`.
